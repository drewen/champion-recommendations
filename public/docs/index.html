get /recommendations/{playerId}/{region}
Recommendations by Player Id (recommendationsPlayerIdRegionGet)
The Recommendations endpoint provides summoners their recommendations based on similarities with other players who have used this system, as well as with current challenger-level players.\n

Path parameters

playerId (required)
Path Parameter — Summoner's player id
region (required)
Path Parameter — League of Legends region (eg. na, euw, kr)

Return type

array[Champion]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

A list of champion recommendations

400

Client error, review your usage of this endpoint.

404

Summoner or data not found. Make sure you have the correct playerId and region

500

Server error, contact administrator
Up 
get /summoner/{name}/{region}
Summoner by Name (summonerNameRegionGet)
The Summoner endpoint returns information about a summoner by their name and region. This will also add them their information to the recommendation engine for later use with other players.\n

Path parameters

name (required)
Path Parameter — Summoner's display name
region (required)
Path Parameter — League of Legends region (eg. na, eune, euw)

Return type

Summoner

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

A summoner

400

Client error, review your usage of this endpoint.

404

Summoner or data not found. Make sure you have the correct playerId and region

500

Server error, contact administrator
